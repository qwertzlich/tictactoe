# Unschlagbares TicTacToe (Minimax AI)

Ein klassisches TicTacToe Spiel, geschrieben in Python. Das Besondere daran ist, dass du gegen eine künstliche Intelligenz spielst, die den **Minimax-Algorithmus** implementiert. Dadurch berechnet die KI alle möglichen Spielausgänge im Voraus und ist buchstäblich unschlagbar. Das beste mögliche Ergebnis für dich als menschlichen Spieler ist ein Unentschieden!

## Projekt-Struktur
- `st.py`: Das Haupt-Interface für das Web-Frontend, geschrieben mit [Streamlit](https://streamlit.io/).
- `npc.py`: Die Logik der KI sowie der Minimax-Algorithmus.

## Wie spiele ich es lokal?
Falls du das Spiel auf deinem eigenen Computer starten willst, benötigst du Python. Führe dazu einfach in deinem Terminal aus:
```bash
pip install streamlit numpy
streamlit run st.py
```
